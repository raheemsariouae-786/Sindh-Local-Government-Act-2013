# Chapter X — Local Taxation

سندھ لوکل گورنمنٹ ایکٹ 2013 کا دسواں باب کونسل کی ٹیکس عائد کرنے، وصول کرنے اور اس کے قواعد و ضوابط سے متعلق دفعات پر مشتمل ہے — لوکل سیس، شیڈول V کے ٹیکس، ماڈل ٹیکس شیڈول، ٹیکس کے سلسلے میں حکومتی ہدایات، وصولی کا طریقہ کار، تنخواہوں سے کٹوتی، اور تشخیص کے خلاف درخواستیں۔
