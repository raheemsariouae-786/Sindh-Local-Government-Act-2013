# Official Digital Legal Reference Portal

A comprehensive public legal reference project for the Sindh Local Government Act 2013.

This project is designed for:

- Councillors
- Chairmen
- Vice Chairmen
- Lawyers
- Judges
- Government Officers
- Researchers
- Students
- Journalists
- Citizens
